# una-algprog-lista04
**Operadores lógicos** são usados para **combinar ou modificar condições booleanas (verdadeiro ou falso)** dentro de estruturas como `if`.

**Principais operadores:**

* **E (AND)**: a condição só é verdadeira se **todas** as partes forem verdadeiras.
* **OU (OR)**: a condição é verdadeira se **pelo menos uma** das partes for verdadeira.
* **NÃO (NOT)**: **inverte** o valor lógico de uma condição (verdadeiro vira falso e vice-versa).

**Exemplo:**

se (idade >= 18) e (temCarteira = verdadeiro) entao
   escreva("Pode dirigir")
fimse

Nesse caso, o `if` só executa se **as duas condições forem verdadeiras**.
Assim, os operadores lógicos permitem criar **condições mais complexas em um único `if`**, combinando várias verificações ao mesmo tempo.
